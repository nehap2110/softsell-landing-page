# Architecture

## Tech Stack
- React
- Vite
- Tailwind CSS
- JavaScript

## Project Structure
The application follows a component-based architecture using React. UI sections are separated into reusable components for better scalability and maintainability.

## Deployment
The project is deployed using Vercel for fast and optimized frontend hosting.

## Design Decisions
- Vite was selected for fast build performance.
- Tailwind CSS was used for rapid UI development and responsiveness.
- React was used for reusable and scalable frontend components.
